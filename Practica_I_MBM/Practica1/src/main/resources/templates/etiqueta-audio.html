Etiqueta <audio>

Descripción y significado

La etiqueta <audio> se utiliza para incrustar contenido de sonido en un documento HTML, como música, efectos sonoros o grabaciones de voz.

  • Sintaxis: Apertura <audio> y cierre </audio>. Adentro lleva <source> para indicar el archivo.
  • Atributos comunes: controls (botones de play/pausa), autoplay y loop.

Ejemplo de código

    <source src="/supieran.mp3" type="audio/mpeg">
</audio>
