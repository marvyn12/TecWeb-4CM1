Etiqueta <a>

Descripción y significado

La etiqueta <a> (del inglés anchor o ancla) se utiliza para crear enlaces o hipervínculos. Es la esencia de la web, ya que permite navegar de una página a otra, descargar archivos o saltar a diferentes partes de un mismo documento.

  • Sintaxis: Apertura <a> y cierre </a>. El texto que el usuario puede clickear va en medio.
  • Atributos comunes: href es obligatorio (indica la ruta o URL destino). target="_blank" se usa mucho para que el enlace se abra en una pestaña nueva.

Ejemplo de código

<p>
    Puedes buscar más información en
    <a href="https://www.google.com" target="_blank">Google</a>.
</p>

Ejemplo renderizado

Puedes buscar más información en Google.
