Etiqueta <button>

Descripción y significado

La etiqueta <button> crea un botón interactivo. Se usa normalmente dentro de los formularios para enviar datos, pero con JavaScript también se puede usar para activar acciones en cualquier parte de la página.

  • Sintaxis: Apertura <button> y cierre </button>. Adentro puedes poner texto o incluso imágenes/iconos.
  • Atributos comunes: type. Si está dentro de un formulario, debe ser type="submit" para que envíe los datos. Si es un botón genérico, usa type="button".

Ejemplo de código

<button type="submit">Enviar Datos</button>
<button type="button" onclick="alert('¡Hola!')">Saludar</button>
