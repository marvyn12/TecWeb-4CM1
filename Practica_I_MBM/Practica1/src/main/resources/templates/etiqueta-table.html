Etiqueta <table>

Descripción y significado

La etiqueta <table> es el contenedor principal para crear una tabla de datos. Sirve para organizar información en filas y columnas, igual que en un archivo de Excel.

  • Sintaxis: Apertura <table> y cierre </table>. Por sí sola no muestra nada, necesita llevar filas (tr) y celdas (td, th) adentro.
  • Nota: Hace muchos años se usaban para diseñar la estructura de toda la página web, pero hoy en día es una pésima práctica; solo deben usarse para mostrar datos tabulares.

Ejemplo de código

<table border="1">
    <tr>
        <th>Nombre</th>
        <th>Edad</th>
    </tr>
    <tr>
        <td>Juan</td>
        <td>25</td>
    </tr>
</table>
