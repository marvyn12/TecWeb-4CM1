Etiqueta <header>

Descripción y significado

La etiqueta <header> representa el encabezado introductorio de una página o de una sección específica. Suele contener el logo, el título principal y, a veces, la barra de navegación.

Ejemplo de código

<header>
    <h1>Mi Sitio Web</h1>
    <p>El mejor lugar de internet</p>
</header>
