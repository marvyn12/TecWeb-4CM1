Etiqueta <section>

Descripción y significado

La etiqueta <section> agrupa contenido temático de tu página que está relacionado entre sí. Imagínalo como los capítulos de un libro (ej. una sección para "Contacto", otra para "Nuestros Servicios").

Ejemplo de código

<section>
    <h2>Nuestros Servicios</h2>
    <p>Hacemos páginas web increíbles y sistemas robustos.</p>
</section>

Ejemplo renderizado

Nuestros Servicios

Hacemos páginas web increíbles y sistemas robustos.
