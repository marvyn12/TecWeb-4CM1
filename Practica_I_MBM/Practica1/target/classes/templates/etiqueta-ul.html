Etiqueta <ul>

Descripción y significado

La etiqueta <ul> (del inglés unordered list) se utiliza para crear una lista de elementos donde el orden no importa. Por defecto, los navegadores muestran estos elementos con viñetas (puntitos).

  • Sintaxis: Apertura <ul> y cierre </ul>. Adentro de esta etiqueta solo deben ir elementos <li>.
  • Atributos comunes: Principalmente class e id. El estilo de la viñeta (círculo, cuadrado, etc.) se cambia con CSS.

Ejemplo de código

<h3>Ingredientes para hotcakes:</h3>
<ul>
    <li>Harina</li>
    <li>Leche</li>
    <li>Huevo</li>
</ul>
