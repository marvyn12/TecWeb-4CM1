Etiqueta <nav>

Descripción y significado

La etiqueta <nav> (del inglés navigation) envuelve los enlaces de navegación más importantes de tu sitio. No debes meter cualquier enlace aquí, solo el menú principal, el índice o la barra lateral.

Ejemplo de código

<nav>
    <a href="/inicio">Inicio</a> |
    <a href="/nosotros">Nosotros</a> |
    <a href="/contacto">Contacto</a>
</nav>
