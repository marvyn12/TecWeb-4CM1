Etiqueta <tr>

Descripción y significado

La etiqueta <tr> (del inglés table row) define una fila horizontal dentro de una tabla. Es como el renglón de tu cuaderno.

  • Sintaxis: Apertura <tr> y cierre </tr>. Siempre debe ir dentro de una <table> (o de un tbody/thead).
  • Contenido: Adentro de un tr no puedes poner texto suelto, solo puedes poner etiquetas de celda (<td> o <th>).

Ejemplo de código

<table border="1">
    <!-- Esta es una fila completa con dos celdas -->
    <tr>
        <td>Dato 1</td>
        <td>Dato 2</td>
    </tr>
</table>
