Etiqueta <input>

Descripción y significado

La etiqueta <input> es el campo de entrada de datos más versátil en HTML. Dependiendo de su tipo, puede ser un cuadro de texto, una casilla de verificación, un selector de color o un botón para subir archivos.

  • Sintaxis: Es una etiqueta vacía, solo tiene apertura <input>.
  • Atributos comunes: type (define el tipo de entrada, ej. "text", "password", "email", "checkbox"), name y placeholder (texto fantasma de ayuda).

Ejemplo de código

<input type="text" placeholder="Escribe tu nombre">
<input type="password" placeholder="Contraseña">
<input type="color">
