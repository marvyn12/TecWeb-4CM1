Etiqueta <ol>

Descripción y significado

La etiqueta <ol> (del inglés ordered list) crea una lista donde el orden numérico o alfabético sí es importante. Por defecto, los navegadores enumeran los elementos (1, 2, 3...).

  • Sintaxis: Apertura <ol> y cierre </ol>. Al igual que la anterior, adentro solo lleva etiquetas <li>.
  • Atributos comunes: type (para cambiar entre números, letras o números romanos, ej. type="A") y start (para decirle desde qué número empezar a contar).

Ejemplo de código

<h3>Pasos para cocinar:</h3>
<ol>
    <li>Mezclar los ingredientes.</li>
    <li>Calentar el sartén.</li>
    <li>Servir con miel.</li>
</ol>
