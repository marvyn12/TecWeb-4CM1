Etiqueta <li>

Descripción y significado

La etiqueta <li> (del inglés list item) representa un elemento individual dentro de una lista. Siempre debe ser "hijo" directo de una etiqueta <ul> (lista desordenada) o <ol> (lista ordenada).

  • Sintaxis: Apertura <li> y cierre </li>.
  • Dato curioso: Adentro de un <li> puedes meter casi cualquier cosa: párrafos, imágenes, enlaces, ¡e incluso otra lista completa para hacer un submenú!

Ejemplo de código

<ul>
    <li>Este es un simple elemento de lista.</li>
    <li>
        Este elemento tiene un <a href="#">enlace</a> adentro.
    </li>
</ul>

Ejemplo renderizado

  • Este es un simple elemento de lista.
  • Este elemento tiene un enlace adentro.
