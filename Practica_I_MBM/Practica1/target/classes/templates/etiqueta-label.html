Etiqueta <label>

Descripción y significado

La etiqueta <label> representa una etiqueta de texto descriptiva para un elemento de formulario. Ayuda muchísimo a la accesibilidad, ya que si haces clic en el texto del label, automáticamente se selecciona el cuadro de texto asociado.

  • Sintaxis: Apertura <label> y cierre </label>.
  • Atributos comunes: for. El valor de este atributo debe ser exactamente igual al atributo id del input que quiere controlar.

Ejemplo de código

<label for="usuario">Nombre de usuario:</label>
<input type="text" id="usuario">
