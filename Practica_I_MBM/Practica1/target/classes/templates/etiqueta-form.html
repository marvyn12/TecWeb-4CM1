Etiqueta <form>

Descripción y significado

La etiqueta <form> es el contenedor principal que agrupa elementos interactivos (como cuadros de texto o botones) para recopilar información del usuario y enviarla a un servidor.

  • Sintaxis: Apertura <form> y cierre </form>.
  • Atributos comunes: action (hacia qué URL se envían los datos) y method (cómo se envían los datos, generalmente GET o POST).

Ejemplo de código

<form action="/procesar-datos" method="POST">
    <!-- Aquí adentro van los inputs y botones -->
</form>

Ejemplo renderizado

Un formulario por sí solo no tiene apariencia visual. Es una caja invisible que prepara los datos para ser enviados.
