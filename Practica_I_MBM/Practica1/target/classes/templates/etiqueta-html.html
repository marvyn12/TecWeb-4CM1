Etiqueta <html>

Descripción y significado

La etiqueta <html> es el elemento raíz (root) de cualquier página web. Se le conoce como el "elemento de nivel superior" porque envuelve absolutamente todo el contenido de tu sitio. Básicamente, es la etiqueta que le avisa al navegador: "Todo lo que está aquí adentro es código HTML, procésalo como tal".

  • Sintaxis: Requiere una etiqueta de apertura <html> justo al inicio (inmediatamente después de la declaración del Doctype) y una etiqueta de cierre </html> hasta el mero final del documento.
  • Atributos comunes: El más vital es lang, que indica el idioma (ej. lang="es"). Otro muy usado aquí es xmlns.

Ejemplo de código

<!DOCTYPE html>
<html lang="es">
    <!-- Todo el contenido de tu sitio web (head y body) vive aquí adentro -->
</html>

Ejemplo renderizado

La etiqueta raíz no tiene una representación gráfica ni estilos visuales propios por defecto. Su "renderizado" es, en esencia, todo el lienzo en blanco de la ventana del navegador donde vivirán el resto de las etiquetas.
