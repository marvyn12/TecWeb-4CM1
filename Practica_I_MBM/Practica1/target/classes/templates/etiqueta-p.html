Etiqueta <p>

Descripción y significado

La etiqueta <p> (del inglés paragraph) se utiliza para definir un párrafo de texto. Es un elemento de bloque, lo que significa que siempre comienza en una nueva línea y ocupa todo el ancho disponible.

  • Sintaxis: Requiere etiqueta de apertura <p> y de cierre </p>.
  • Atributos comunes: Principalmente atributos globales como class, id, y style.

Ejemplo de código

<p>
    Este es un párrafo de ejemplo.
    Puedes poner mucho texto aquí adentro y el navegador
    lo agrupará como un solo bloque.
</p>
