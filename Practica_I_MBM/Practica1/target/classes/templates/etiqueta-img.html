Etiqueta <img>

Descripción y significado

La etiqueta <img> se utiliza para incrustar una imagen en una página web. A diferencia de otras etiquetas, esta no envuelve ningún contenido de texto, sino que "llama" a un archivo gráfico para mostrarlo en pantalla.

  • Sintaxis: Es una "etiqueta vacía", lo que significa que solo tiene apertura y no lleva etiqueta de cierre.
  • Atributos comunes: src (obligatorio, indica la ruta de la imagen) y alt (obligatorio para accesibilidad).

Ejemplo de código

<img src="/komanderCereal.jpeg" alt="El Komander Comiendo Cereal" width="300">
