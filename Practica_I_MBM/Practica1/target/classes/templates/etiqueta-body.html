Etiqueta <body>

Descripción y significado

La etiqueta <body> representa el contenido principal de un documento HTML. Todo lo que el usuario ve y con lo que interactúa en la pantalla (textos, imágenes, enlaces, botones) debe ir obligatoriamente aquí adentro.

  • Sintaxis: Va justo después de cerrar el </head>. Solo puede haber un body por documento.
  • Atributos comunes: Anteriormente se usaban atributos para color de fondo, pero hoy en día todo eso se maneja con class, id y CSS.

Ejemplo de código

<body>
    <h1>Bienvenido a mi web</h1>
    <p>Todo este texto sí es visible para el usuario.</p>
</body>

Ejemplo renderizado

Bienvenido a mi web

Todo este texto sí es visible para el usuario.
