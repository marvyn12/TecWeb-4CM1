# 🌐 Tecnologías para el Desarrollo de Aplicaciones Web

Bienvenido a mi repositorio de la unidad de aprendizaje de Tecnologías para el desarrollo de aplicaciones Web. Este espacio contiene proyectos, prácticas y ejercicios enfocados en la creación de aplicaciones web modernas, abarcando desde el diseño de interfaces hasta la lógica del lado del servidor.

## 🎯 Objetivo del Repositorio
El propósito de esta colección es documentar mi capacidad para desarrollar aplicaciones Web con base en las tecnologías HTML, XML, Java, JavaScript, AJAX, frameworks y la conexión a bases de datos.

## 📚 Áreas de Conocimiento
Los proyectos documentados en este repositorio abarcan los siguientes temas principales:

* **Fundamentos de la Web:** Arquitectura Cliente/Servidor, protocolo TCP/IP, HTTP y estándares de la Web.
* **Estructura y Contenido (HTML):** Diseño de contenido web estático y dinámico utilizando elementos de HTML, tablas y formularios.
* **Diseño y Maquetado (CSS):** Diseño responsivo, modelo de cajas, posicionamiento moderno (GridLayout, Flexbox) y propiedades avanzadas de CSS3.
* **Interactividad (JavaScript y AJAX):** Programación del lado del cliente, modelo de eventos, notación JSON, consumo de datos asíncronos (AJAX) y uso de frameworks.
* **Lógica de Servidor (Java Servlets):** Programación del lado del servidor (Back-End), ciclo de vida de los Servlets, manejo de sesiones y cookies.
* **Manejo de Datos (XML):** Diseño de contenido web basado en documentos y reglas XML, Modelos de Objetos del Documento (DOM) y Esquemas XML.

## 📂 Estructura del Repositorio
El repositorio está organizado por prácticas y proyectos. Cada carpeta interna contiene el código fuente (Front-End y Back-End), la estructura de archivos necesaria y los recursos gráficos utilizados para cada implementación.
